# mxnet-for-jetson-ubuntu20.04_python3.8

Follow the following steps to install the mxnet on jetson nano with ubuntu20.04 that comes with Python 3.8

1. Download the file (it's a big file...I'll upload it soon)
2. Install the following dependencies
sudo apt-get update
sudo apt-get install -y \
                        build-essential \
                        git \
                        libopenblas-dev \
                        libopencv-dev \
                        python3-pip \
                        python-numpy

sudo pip3 install --upgrade \
                        pip \
                        setuptools \
                        numpy
                        
3. $ cd mxnet/python
   $ sudo pip3 install -e .

4. Test the installation
$ python3
import mxnet
mxnet.__version__

This shoould show: '1.8.0'


5. Test gpu:
